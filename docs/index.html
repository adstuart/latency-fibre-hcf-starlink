Fixed assumptions

Well-known physical and industrial constants. We hold them fixed and document the rationale here rather than expose fake-precision sliders.

  • SMF refractive index n = 1.468 — ITU-T G.652.D silica group index at 1550 nm. Applies to SMF only; HCF uses its own n (see next line). All production submarine cables use G.652.D.
  • HCF speed = 0.997 c (n ≈ 1.003) — NANF hollow-core propagation. Light travels through air, not silica. Physics what-if for transoceanic; today's deployments are metro / intra-DC only.
  • Cable lengths — LON↔NYC 6,200 km, LON↔SYD 22,000 km (TeleGeography 2025 Submarine Cable Map; representative production paths).
  • Starlink ISL grid detour = 1.15× — typical real-world grid routing with ~8,100 operational sats × 4 laser ISLs each (April 2026). 1.00× is the press-release geodesic lower bound; 1.10–1.20× is observed reality; 1.30× is polar hand-off worst case.
  • Speed of light c = 299,792.458 km/s — vacuum, exact.
  • ISL hop cost = 1 ms — per-satellite routing overhead on top of geometric propagation.
